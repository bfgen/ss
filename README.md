# Symfony Skeleton

---

### Setup

```shell
$ make setup
```

> It will copy example env into .env,
> run docker build,
> up containers in background
> and run composer install

---

### Run

```shell
$ make up
```

Background run

```shell
$ make up-d
```

### Stop

```shell
$ make down
```

---

### Enter app container

```shell
$ make app
```

---

#### Docker rebuild

```shell
$ make build
```

---

#### Composer install

```shell
$ make vendor
```
