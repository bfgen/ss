# Symfony Skeleton

---

### Setup

```shell
$ make setup
```

> It will run docker build and copy example env into .env

---

### Run

```shell
$ make up
```

Background run

```shell
$ make up-d
```

### Stop

```shell
$ make down
```

---

### Enter app container

```shell
$ make app
```

---

#### Docker rebuild

```shell
$ make build
```
